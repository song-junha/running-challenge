# 🏃‍♂️ 러닝 챌린지 사용 가이드

## 📱 주요 기능

### 1️⃣ Strava 연동
- **홈 화면** 또는 **등록 페이지**에서 "연동" 버튼 클릭
- Strava 계정으로 로그인하여 앱 연결
- 연동 후 자동으로 러닝 데이터 동기화 시작

### 2️⃣ 러닝 기록 비교
**홈 화면**에서 기간별 러닝 기록을 비교할 수 있습니다:
- **이번달**: 현재 월의 러닝 기록
- **저번달**: 이전 월의 러닝 기록
- **올해**: 올해 전체 러닝 기록
- **작년**: 작년 전체 러닝 기록

각 사용자의 총 거리, 활동 수, 총 시간이 카드 형태로 표시됩니다.

### 3️⃣ 데이터 동기화
- **동기화 버튼**: 최근 1개월 데이터를 Strava에서 가져옵니다
- 새로운 러닝 기록을 추가한 후 동기화 버튼을 눌러 업데이트하세요

### 4️⃣ 최근 활동 조회
**활동 탭**에서 모든 사용자의 최근 러닝 활동을 확인할 수 있습니다:
- 사용자별 최근 30개 활동 표시 (전체 최대 200개)
- 날짜 내림차순 정렬
- 활동 카드 클릭 시 상세 정보 모달 표시

**활동 상세 정보**:
- 러닝 경로 맵 (Mapbox)
- 속도/심박수/케이던스 차트
- 거리, 시간, 평균 속도, 고도 등 상세 데이터

### 5️⃣ 대회 관리
**대회 탭**에서 러닝 대회를 등록하고 관리할 수 있습니다:

**대회 등록**:
- "등록" 버튼 클릭
- 날짜와 대회명 입력
- 참가자 추가 (선택사항)

**대회 수정/삭제**:
- 관리자 비밀번호 입력 필요
- "수정" 버튼으로 대회 정보 변경
- "삭제" 버튼으로 대회 제거

**대회 필터링**:
- "예정된 대회" / "지난 대회" 탭으로 필터링
- 검색창에서 대회명 또는 참가자 이름으로 검색

**대회 결과**:
- 대회 기간 동안의 러닝 기록이 자동으로 집계
- 참가자별 총 거리, 활동 수, 총 시간 표시
- 순위는 총 거리 기준

### 6️⃣ 사용자 관리
**등록 탭**에서 사용자 정보를 관리할 수 있습니다:

**닉네임 변경**:
- 자신의 닉네임은 비밀번호 없이 변경 가능
- 다른 사용자의 닉네임 변경 시 관리자 비밀번호 필요
- 닉네임 변경 시 기존 대회 참가자 이름도 자동 업데이트

**사용자 삭제**:
- 관리자 비밀번호 입력 필요
- 삭제 시 해당 사용자의 모든 활동 데이터도 함께 삭제

### 7️⃣ 개인 기록 조회
**홈 화면**의 사용자 카드를 클릭하면 개인 기록 모달이 표시됩니다:
- 최장 거리 러닝
- 가장 긴 시간 러닝
- 최고 속도 러닝
- 각 기록의 상세 정보 확인 가능

## 🔐 관리자 기능

관리자 비밀번호가 필요한 기능:
- ❌ 사용자 삭제 (항상 필요)
- ❌ 다른 사용자 닉네임 수정 (자신의 닉네임은 불필요)
- ❌ 대회 수정/삭제 (항상 필요)
- ✅ 대회 등록 (비밀번호 불필요)

## 📊 수집되는 데이터

각 러닝 활동마다 다음 데이터가 저장됩니다:
- 기본: 거리, 시간, 속도, 고도
- 심박수: 평균 심박수, 최대 심박수
- 케이던스: 평균 케이던스
- 환경: 기온
- 기타: 칼로리, 운동 강도 점수

## 💡 사용 팁

1. **첫 연동 시**: 데이터 동기화에 시간이 걸릴 수 있습니다 (5년치 데이터)
2. **정기 동기화**: 새로운 러닝 후 동기화 버튼으로 최신 데이터 업데이트
3. **대회 참가**: 대회 등록 시 참가자를 미리 추가해두면 자동으로 결과 집계
4. **닉네임 설정**: Strava 이름 대신 원하는 닉네임 사용 가능
5. **데이터 보안**: 비공개 활동은 수집되지 않습니다

## 🚀 시작하기

1. Strava 계정으로 연동
2. 데이터 동기화 완료 대기
3. 홈 화면에서 러닝 기록 확인
4. 대회를 만들어 친구들과 경쟁!
